(person != person2 / person == person3) 위 내용에 덧붙이면, let person2 = {...person, address: 'seoul'} // person을 복사한 후, address라는 새 값을 추가하여 객체를 생성한다. let person2 = {...person, name: 'seoul'} // person의 name의 값을 seoul로 바꾸고 객체를 생성한다. let animal = {legs: 2, arms: 2} let person2 = {...person, ...animal} // person과 animal 두 객체의 값들을 모두 복사하여 새 객체를 생성한다. 위 방법 역시 배열에도 사용가능하다. -------------------------------------------------------------- 4. 삼항 연산자 (고급 프로그래밍 때 배운 것) let person = {name:'kitae', age:12} if(person) { console.log(person.name) } else { console.log("there is no person") } 위의 코드를 아래 삼항 연산자를 활용하여 표현할 수 있다. console.log(person?person.name:"there is no person") ======================================================================================================================================================== <<>> 이전엔 state를 class에서만 사용가능했다. 하지만 hook이란 것이 개발되면서 function에서도 state를 사용할 수 있게 되었다. 또한, life cycle 역시 class에서만 다뤘었지만, hook을 통해 function에서도 다룰수 있게 되어서 function이 class에 견줄 수 있게 되었다. hook은 기본적으로 'use이름'으로 구성되어 있다. ex)useState, useEffect hook은 class외에서 state 값을 사용하게 하는 역할을 하며, 사용 방법은 다음과 같다. - import {useState} from 'react' 를 새로 import 해야한다 - useState()는 배열이므로 아래와 같이 사용한다. - const [상태이름, set상태이름] = useState(첫 상태 값);
